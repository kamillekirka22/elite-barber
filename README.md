# 💈 Barbearia Elite - Landing Page

Este projeto é uma landing page desenvolvida para uma barbearia fictícia, com foco em apresentação de serviços e agendamento online.

A aplicação foi criada com o objetivo de praticar desenvolvimento front-end moderno utilizando React, TypeScript e Tailwind CSS.

---

## 🚀 Funcionalidades

- ✅ Página responsiva
- ✅ Seções institucionais (Sobre, Serviços, Galeria, Depoimentos)
- ✅ Formulário de agendamento
- ✅ Integração com EmailJS para envio de agendamentos por e-mail
- ✅ Feedback visual ao usuário após envio
- ✅ Interface moderna e estilizada

---

## 🛠️ Tecnologias utilizadas

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- EmailJS

---

## 📩 Como funciona o agendamento

O usuário preenche o formulário com:

- Nome
- Telefone
- Data
- Horário

Ao enviar, os dados são enviados automaticamente para o e-mail configurado via EmailJS.
